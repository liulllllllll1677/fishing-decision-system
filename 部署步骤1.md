===========================================
步骤 1：创建 GitHub 仓库
===========================================

1. 打开浏览器，访问：https://github.com/new
2. 如果未登录，点击 "Sign in" 登录你的 GitHub 账号
3. 填写仓库信息：
   - Repository name（仓库名）：fishing-master-v2
   - Description（描述）：钓鱼大师决策系统 - 专业级垂钓决策平台
   - 选择 Public（公开）
4. **不要**勾选 "Add a README file"
5. 点击绿色按钮 "Create repository"

创建完成后，GitHub会显示一个页面，请复制仓库地址。
仓库地址格式通常是：https://github.com/你的用户名/fishing-master-v2
