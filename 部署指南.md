## 🚀 快速部署（5分钟获得永久网址）

### 方法一：Netlify Drop（最简单，推荐）⭐

1. 打开这个网址：https://app.netlify.com/drop
2. 直接将 **整个文件夹** 拖到浏览器窗口中
3. 等待几秒钟，自动部署完成！
4. 你会得到一个永久网址，例如：
   `https://peaceful-yak-123456.netlify.app`

✅ **优点：**
- 无需注册账号（如果已登录 Netlify）
- 完全免费
- 全球 CDN 加速
- 支持自定义域名
- **推荐新手使用**

---

### 方法二：GitHub Pages（永久免费，适合开发者）⭐⭐

#### 步骤1：创建 GitHub 仓库
1. 访问 https://github.com/new
2. 仓库名填写：`fishing-master-v2`
3. 设置为 Public（公开）
4. 点击 "Create repository"

#### 步骤2：上传代码
在仓库页面点击 "uploading an existing file"，然后：
- 拖拽所有文件（`index.html`、`fishing-data.js`、`README.md`）到页面
- 填写提交信息："Initial commit: 钓鱼大师决策系统 v2.0"
- 点击 "Commit changes"

#### 步骤3：启用 GitHub Pages
1. 进入仓库 → 点击 "Settings"
2. 左侧菜单找到 "Pages"
3. 在 "Source" 下拉菜单选择 "Deploy from a branch"
4. Branch 选择 "main"，文件夹选择 "/ (root)"
5. 点击 "Save"

#### 步骤4：等待部署
等待 1-2 分钟后，访问：
`https://你的用户名.github.io/fishing-master-v2`

✅ **优点：**
- GitHub 官方服务，稳定可靠
- 完全永久免费
- 可以使用自定义域名
- 支持版本控制
- **推荐给有Git基础的用户**

## 📱 使用说明

### 如何使用决策系统

1. **选择目标鱼种**
   - 左侧第一个选项，选择你今天想钓的鱼
   - 包含15+种常见和珍稀鱼类

2. **选择目标鱼体大小**
   - 根据你所在钓场的情况选择
   - 从小型（< 0.5斤）到巨型（> 15斤）

3. **选择当前季节**
   - 选择当前的季节时段
   - 影响饵料选择和钓法策略

4. **选择天气状况**
   - 选择今天的天气
   - 影响鱼的活跃度和钓位选择

5. **选择钓场类型**
   - 选择你今天去的钓场
   - 影响钓位和钓法策略

6. **选择出钓时间段**
   - 选择你出钓的时间
   - 影响钓层和钓位选择

7. **点击"生成作钓方案"**
   - 系统会自动生成完整的作钓方案
   - 包含线组、饵料、钓位、策略等所有信息

### 查看结果

右侧会显示：
- ✅ 钓况分析
- 🔧 线组搭配（主线、子线、钩号、浮漂）
- 🐛 饵料推荐（主饵、副饵、配比、提示）
- 📍 钓位选择（推荐位置、水深、最佳时段）
- 🎯 钓法策略（核心策略、钓层、天气应对）
- ⭐ 老钓友经验之谈
